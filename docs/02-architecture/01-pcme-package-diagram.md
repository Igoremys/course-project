## 1. Назначение диаграммы
Диаграмма визуализирует структуру пакетов клиент-серверной системы, построенной на основе архитектурного паттерна **PCMEF (Presentation-Control-Mediator-Entity-Foundation)**. Она демонстрирует строгое разделение слоёв, направленность зависимостей сверху вниз и взаимодействие мобильного клиента (Kotlin/Android) с серверной частью (Java/Spring Boot) через REST API.
