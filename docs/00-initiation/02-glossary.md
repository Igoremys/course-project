## Бизнес-глоссарий
| Термин | Определение |
|---|---|
| Портфолио | Коллекция отобранных фотографий, структурированная по темам/датам |
| Альбом | Логическая группа внутри портфолио |
| Метаданные | EXIF, теги, описание, авторство, права использования |
| Кэширование | Локальное хранение данных через Room для работы без сети |
| JWT | JSON Web Token для stateless-аутентификации |
| Эндпоинт | URL-маршрут REST API |
| Синхронизация | Процесс согласования локальных изменений с сервером |
| Превью | Уменьшенная копия изображения для быстрого отображения |
| PCMEF | Архитектурный паттерн: Presentation-Control-Mediator-Entity-Foundation |
| REST API | Архитектурный стиль взаимодействия клиента и сервера через HTTP/JSON |
| Room Database | Библиотека персистентности Android для локального кэширования |
| Пагинация | Механизм разделения больших списков данных на страницы |
| CRUD-операции | Базовый набор функций: Create, Read, Update, Delete |
| Conflict Resolution | Стратегия разрешения конфликтов при оффлайн-синхронизации |
| StateFlow | Kotlin-поток для реактивного обновления UI из ViewModel |
